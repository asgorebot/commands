# Commands
A list of commands for the bot Asgore

# Prefix
> [!IMPORTANT] 
> All commands will be under the default prefix: `;` unless you have set a custom prefix

## Moderation
```
;ban (user) (reason)
;unban (user) (reason)
;timeout (user) (duration) (reason)
```

## Log
```
;log (type) (channel)
```

## setup
```
;setup join
```

## Fun
```
;steam (steamid)
;dad
```
